# LinkCutter — promo landing

Промо-лендинг Telegram-бота [@LinkCutterBot](https://t.me/LinkCutterBot), который убирает
трекинговые параметры из ссылок.

🌐 **Live:** https://ttalapov.github.io/linkcutter/

- Статика без сборки: HTML + CSS + vanilla JS, три языка (uk/ru/en) переключаются на клиенте.
- Публикуется через GitHub Pages из корня репозитория.
- Код бота — в отдельном приватном репозитории `linkcutter-bot`.

Локальный просмотр: открыть `index.html` в браузере.
